DALL-E 3

Guía Completa de Prompts DALL-E 3

Marzo 2026 · 10 min de lectura · DALL-E 3 · ChatGPT · OpenAI · Actualizado marzo 2026

DALL-E 3 de OpenAI es único entre los generadores de imágenes con IA. A diferencia de Midjourney, Stable Diffusion o Flux, está profundamente integrado con ChatGPT, lo que significa que puedes describir tus imágenes en lenguaje natural conversacional. Además, DALL-E 3 es actualmente el mejor modelo para renderizar texto legible en las imágenes generadas. Esta guía te enseña a aprovechar estas capacidades únicas.

Cómo Acceder a DALL-E 3

DALL-E 3 está disponible a través de varios canales:

  • ChatGPT Plus: 20$/mes — la forma más sencilla, con interfaz conversacional
  • ChatGPT Enterprise / Team: Para equipos y empresas
  • API de OpenAI: Acceso programático, tarificación por token
  • Microsoft Copilot: Versión gratuita limitada (antes Bing Image Creator)

El acceso vía ChatGPT es recomendado para la mayoría de usuarios ya que ChatGPT actúa como asistente de prompting, reformulando automáticamente tus solicitudes para optimizar los resultados.

El Enfoque de Lenguaje Natural

DALL-E 3 fue diseñado para comprender el lenguaje natural conversacional. No necesitas sintaxis especial, ponderación ni parámetros. Simplemente escribe lo que quieres ver, como si estuvieras describiendo la imagen a alguien:

A cozy reading nook by a large window on a rainy day. There's a comfortable armchair with a knitted blanket, a steaming cup of tea on a small side table, and a bookshelf filled with old leather-bound books. The light is soft and warm, filtering through rain-streaked glass. The style should be like a watercolor illustration with soft, muted colors.

Este prompt en lenguaje natural producirá un resultado muy cercano a la descripción. DALL-E 3 comprende el contexto, la atmósfera y las relaciones espaciales entre los elementos mejor que cualquier otro modelo cuando recibe frases completas.

Estructura recomendada

Para resultados óptimos, estructura tu prompt así:

  1. Sujeto principal: Lo que está en el centro de la imagen
  2. Contexto / escenario: Dónde se desarrolla la escena
  3. Iluminación y ambiente: La atmósfera deseada
  4. Estilo artístico: El acabado visual (fotografía, ilustración, pintura, etc.)
  5. Detalles específicos: Colores, composiciones, elementos adicionales

Renderizado de Texto en Imágenes

Esta es la fortaleza más distintiva de DALL-E 3. A diferencia de Midjourney y Stable Diffusion, que fallan regularmente al renderizar texto legible, DALL-E 3 maneja relativamente bien las palabras cortas.

Buenas prácticas para el texto

  • Pon el texto entre comillas: a coffee mug with the text "Good Morning" written on it
  • Limítate a 1-5 palabras: Más allá, los errores de renderizado son frecuentes
  • Especifica el estilo tipográfico: in bold sans-serif letters, in elegant script font
  • Indica la ubicación: text centered on the front of the mug

La Integración con ChatGPT

Cuando usas DALL-E 3 a través de ChatGPT, se produce un proceso de reformulación automática:

  1. Escribes tu solicitud en lenguaje natural
  2. ChatGPT interpreta tu solicitud y genera un prompt detallado y optimizado
  3. Este prompt reformulado se envía a DALL-E 3
  4. La imagen se genera y se muestra con el prompt utilizado

Esta reformulación es generalmente beneficiosa — ChatGPT añade detalles de composición, iluminación y estilo que no has especificado. Sin embargo, si quieres control total, puedes indicar:

Genera una imagen con exactamente este prompt, sin modificarlo: [tu prompt aquí]

Consejo: el modo iterativo

Una de las ventajas únicas de la integración con ChatGPT es la iteración conversacional. Después de una generación, puedes decir:

  • “Haz la iluminación más cálida y añade más niebla”
  • “Mantén el mismo estilo pero reemplaza el gato por un perro”
  • “Rehaz la misma imagen pero en formato paisaje 16:9”
  • “Me gusta el estilo, pero hazlo más oscuro y dramático”

Política de Contenido

DALL-E 3 tiene la política de contenido más restrictiva entre los grandes generadores. Estas son las limitaciones clave:

  • Sin personalidades reales: Imposible generar retratos de personas reales identificadas
  • Sin contenido violento explícito: Las escenas de violencia gráfica son rechazadas
  • Sin contenido para adultos: Ningún contenido sexualmente explícito
  • Estilos de artistas recientes: DALL-E 3 rechaza copiar el estilo de artistas vivos identificados

Para sortear la restricción sobre estilos de artistas, describe las características visuales en lugar de nombrar al artista: en vez de “in the style of [artista]”, describe “with bold geometric shapes, vibrant primary colors, and thick black outlines”.

Limitaciones y Soluciones

  • Resolución limitada: DALL-E 3 genera en 1024x1024, 1792x1024 o 1024x1792. Usa un upscaler para mayor resolución.
  • Sin control fino: Sin CFG scale, sin sampler, sin seed reproducible. La iteración se hace conversacionalmente.
  • Reformulación automática: ChatGPT modifica tus prompts. Pide explícitamente que no lo haga si es necesario.
  • Consistencia de personajes: Difícil mantener el mismo personaje entre generaciones. Describe los detalles físicos con precisión.

Preguntas Frecuentes

¿Cómo acceder a DALL-E 3?

DALL-E 3 está disponible a través de ChatGPT Plus (20$/mes), ChatGPT Enterprise, la API de OpenAI y Microsoft Copilot (versión gratuita limitada). El acceso vía ChatGPT es el más sencillo ya que ChatGPT reformula automáticamente tus prompts para optimizar los resultados.

¿Puede DALL-E 3 generar texto legible en las imágenes?

Sí, es una de las fortalezas únicas de DALL-E 3. Puede renderizar texto corto (1-5 palabras) de forma bastante fiable en las imágenes. Para mejores resultados, especifica el texto entre comillas en tu prompt y precisa la fuente y el estilo deseados.

¿Por qué ChatGPT modifica mis prompts de DALL-E 3?

ChatGPT actúa como intermediario y reformula tus prompts para mejorar los resultados y cumplir con las políticas de contenido de OpenAI. Puedes ver el prompt modificado en la interfaz. Para controlarlo, indica “Usa exactamente este prompt, no lo modifiques” en tu mensaje.

¿DALL-E 3 soporta prompts negativos?

No, DALL-E 3 no soporta prompts negativos como Stable Diffusion. Para evitar elementos no deseados, formula tu prompt de manera positiva o indica explícitamente lo que no quieres en el texto del prompt (ej: without any text or watermarks).
