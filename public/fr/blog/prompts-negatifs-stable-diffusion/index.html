Prompts Négatifs dans Stable Diffusion : Guide Complet

Mars 2026 · 9 min de lecture · Stable Diffusion · SDXL · Automatic1111 · Mis à jour mars 2026

Les prompts négatifs sont l'un des outils les plus puissants et les moins compris de Stable Diffusion. Ils permettent d'indiquer au modèle ce que vous ne voulez pas voir dans l'image générée — des mains déformées aux filigranes, en passant par les artefacts de compression. Bien utilisés, ils améliorent considérablement la qualité de chaque génération.

Qu'est-ce qu'un Prompt Négatif ?

Dans Stable Diffusion, le prompt négatif est un champ de texte séparé où vous indiquez les éléments, qualités ou styles que vous souhaitez éviter dans l'image générée. Pendant le processus de débruitage, le modèle s'éloigne activement de ces concepts, ce qui produit des images plus propres et plus précises.

Pensez-y comme une boussole : le prompt positif indique la direction souhaitée, et le prompt négatif définit les directions à éviter. Ensemble, ils canalisent la génération vers un résultat précis.

Comment Ça Fonctionne Techniquement

Stable Diffusion utilise un processus appelé Classifier-Free Guidance (CFG) pour équilibrer le prompt positif et le prompt négatif. À chaque étape de débruitage, le modèle calcule deux prédictions :

  1. Une prédiction guidée par votre prompt positif
  2. Une prédiction guidée par votre prompt négatif (ou inconditionnelle si aucun prompt négatif n'est fourni)

Le résultat final est calculé en s'éloignant de la prédiction négative et en se rapprochant de la prédiction positive, selon l'intensité définie par le CFG scale.

L'Interaction avec le CFG Scale

Le CFG scale détermine la force avec laquelle le modèle suit vos prompts. Cette valeur affecte à la fois le prompt positif et le prompt négatif :

CFG Scale Effet Recommandation
1 – 3 Créatif, ignore partiellement les prompts Exploration libre
5 – 7 Équilibre qualité/créativité Usage général (recommandé)
8 – 12 Suit les prompts de près Quand la précision compte
13+ Trop strict, artefacts probables À éviter généralement

Un point clé : un CFG élevé amplifie aussi l'effet des prompts négatifs. Si votre prompt négatif est trop agressif avec un CFG de 12+, vous risquez de produire des résultats étranges où le modèle s'éloigne trop de certains concepts.

Syntaxe et Pondération

Les prompts négatifs supportent la même syntaxe de pondération que les prompts positifs dans Stable Diffusion :

  • (terme:1.4) — pondération forte, le modèle évite activement ce concept
  • (terme:1.0) — pondération normale (par défaut)
  • (terme:0.6) — pondération faible, influence légère
  • [terme] — réduit la pondération de 10%
  • (terme) — augmente la pondération de 10%

Nous recommandons de pondérer les termes de qualité à 1.4 ((worst quality:1.4)) car ils ont l'impact le plus fort sur la qualité générale, tout en gardant les termes spécifiques au contenu à 1.0.

Erreurs Courantes

  1. Prompt négatif trop long. Comme le prompt positif, le prompt négatif a une limite de tokens (77 tokens pour SDXL). Au-delà, les termes supplémentaires sont ignorés. Priorisez les éléments les plus importants.
  2. Mettre des descriptions positives dans le négatif. N'écrivez pas not blurry — écrivez simplement blurry. Le champ négatif est déjà interprété comme "éviter ceci".
  3. Contradictions entre positif et négatif. Si votre prompt positif demande painting et votre prompt négatif contient painting, le modèle reçoit des signaux contradictoires. Le résultat sera incohérent.
  4. Copier des listes sans les adapter. Les prompts négatifs universels sont un bon départ, mais ils doivent être adaptés à votre cas spécifique. Un prompt négatif pour un portrait n'est pas le même que pour un paysage.

Questions Fréquentes

Qu'est-ce qu'un prompt négatif dans Stable Diffusion ?

Un prompt négatif est un champ de texte séparé où vous indiquez les éléments que vous ne voulez PAS voir dans l'image générée. Le modèle s'éloigne activement de ces concepts lors de la génération, améliorant la qualité et réduisant les artefacts.

Le CFG scale affecte-t-il les prompts négatifs ?

Oui, le CFG scale (Classifier-Free Guidance) détermine l'intensité avec laquelle le modèle suit vos prompts positifs ET négatifs. Un CFG de 7 est un bon équilibre. Un CFG trop élevé (15+) peut créer des artefacts, tandis qu'un CFG trop bas (3-4) ignore partiellement vos prompts négatifs.

Quels sont les prompts négatifs universels les plus utiles ?

Les prompts négatifs universels les plus efficaces sont : (worst quality:1.4), (low quality:1.4), blurry, watermark, signature, text, deformed, bad anatomy, bad hands, extra fingers, missing fingers, extra limbs, duplicate, cropped, out of frame, ugly.

Midjourney et Flux supportent-ils les prompts négatifs ?

Midjourney offre un support limité via le paramètre --no (ex: --no text, watermark). Flux ne supporte pas du tout les prompts négatifs. Seul Stable Diffusion propose un champ dédié complet avec support de pondération.
