隐私政策

最后更新：2026年3月 · ImageToPrompt / imagetoprompt.dev

引言

ImageToPrompt (imagetoprompt.dev) 由Othmane Ferhan运营。本隐私政策说明了您使用我们的免费AI图像转提示词生成器时，我们如何收集、使用和保护您的信息。使用ImageToPrompt即表示您同意本政策中描述的做法。

我们致力于保护您的隐私。ImageToPrompt采用隐私优先的设计理念：我们不需要账号，不存储您的图像，仅收集最少量的分析数据。

我们收集的数据

账号和个人信息

ImageToPrompt不需要创建账号或登录即可使用。我们不收集您的姓名、电子邮箱、电话号码或任何个人身份信息，除非您自愿提供（例如订阅我们的可选新闻通讯）。

如果您订阅了我们的新闻通讯，我们仅收集您的电子邮箱地址，仅用于发送新闻通讯。我们不会将此地址出售或分享给第三方用于营销目的。

分析数据

我们使用Google Analytics 4收集匿名的统计数据。这些数据包括：

  • 访问的页面和每页停留时间
  • 设备类型、操作系统和浏览器
  • 国家级地理位置（非精确位置）
  • 会话时长和用户流程
  • 引荐网站

这些数据是匿名的，无法用于识别个人用户。它们仅用于了解工具的使用情况并加以改进。

上传的图像

您的隐私受到保护：我们不存储、保存或保留您上传的任何图像。所有图像处理都是临时的。

  • 所有用户上传的图像直接发送到Claude Vision API（由Anthropic提供）进行分析，并在提示词生成后立即丢弃。
  • 我们不在我们的服务器上存储、保存、缓存或保留任何上传的图像。
  • 我们不将上传的图像用于模型训练、广告、分析或除实时生成请求的文本提示词之外的任何目的。
  • 图像处理完全实时进行。ImageToPrompt不会复制或保留您的图像。
  • Anthropic（Claude API的提供商）作为API调用的一部分处理您的图像。请参阅Anthropic隐私政策了解他们如何处理API数据。

生成的提示词

  • 提示词历史仅存储在您设备的浏览器localStorage中，永远不会离开您的设备。
  • 我们不在我们的服务器上存储生成的提示词。
  • 您可以随时通过清除浏览器本地存储或使用工具中的“清除历史”按钮来清除提示词历史。

Cookie

我们仅将Cookie用于分析目的。我们不使用广告Cookie、跟踪像素或任何第三方营销Cookie。

我们使用的Cookie：

  • _ga — Google Analytics，用于区分用户（2年后过期）
  • _gid — Google Analytics，用于区分用户（24小时后过期）
  • _ga_* — Google Analytics 4会话Cookie

您可以在浏览器设置中禁用Cookie。禁用分析Cookie不会影响您使用ImageToPrompt的能力。

第三方服务

我们使用以下第三方服务来运营ImageToPrompt：

  • Anthropic Claude API — 处理上传的图像以生成提示词。Anthropic隐私政策 →
  • Vercel — 网站托管和基础设施。Vercel隐私政策 →
  • Google Analytics — 匿名使用统计。Google隐私政策 →
  • Upstash Redis — 速率限制（仅存储匿名的基于IP的请求计数，不存储个人数据）。Upstash隐私政策 →

数据安全

  • 所有与ImageToPrompt的连接均使用HTTPS加密。
  • 我们不在服务器上存储个人数据或上传的图像。
  • 由于我们不保留您的图像或个人数据，因此不存在数据泄露暴露您图像的风险。
  • 速率限制数据（匿名的基于IP的请求计数）存储在Upstash Redis中，24小时后自动过期。

儿童隐私

ImageToPrompt不针对13岁以下的用户。我们不会有意收集13岁以下儿童的任何个人信息。如果您认为13岁以下的儿童向我们提供了个人信息，请联系我们，我们将采取措施将其删除。

您的权利

  • 知情权：本隐私政策描述了我们收集的所有数据。我们不收集本文描述之外的个人数据。
  • 选择退出分析的权利：您可以通过安装Google Analytics退出浏览器插件或在浏览器设置中禁用Cookie来禁用Google Analytics。
  • 数据删除权：由于我们不存储您的图像或个人数据，通常没有需要删除的内容。如果您订阅了新闻通讯并希望取消，请通过以下邮箱联系我们。
  • 访问权：您可以通过联系我们请求了解我们持有的关于您的任何数据。

政策变更

我们可能会不时更新本隐私政策。更新时，我们会更新本页面顶部的“最后更新”日期。在变更发布后继续使用ImageToPrompt即表示您接受更新后的政策。我们建议您定期查阅本政策。
