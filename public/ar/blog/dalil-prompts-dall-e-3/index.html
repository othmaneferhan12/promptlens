كيف يعمل DALL-E 3 مع ChatGPT؟

عندما تطلب صورة من ChatGPT، تحدث عملية من خطوتين:

  1. ChatGPT يُعيد صياغة برومبتك: يأخذ وصفك العام ويُحوّله إلى برومبت مفصّل ومحسّن لـ DALL-E 3. هذا يعني أن برومبتك الأصلي ليس ما يراه DALL-E 3 — ChatGPT يُضيف تفاصيل ويُعيد الصياغة.
  2. DALL-E 3 يُولّد الصورة: يستقبل البرومبت المُعاد صياغته ويُنشئ الصورة بناءً عليه.

هذا التكامل ميزة ونقطة ضعف في آنٍ واحد. الميزة: يمكنك الكتابة بأي لغة وبأي مستوى من التفصيل وChatGPT سيُحسّن طلبك. نقطة الضعف: أحياناً يُغيّر ChatGPT نيّتك الأصلية ويُضيف عناصر لم تطلبها.

ChatGPT مقابل DALL-E 3 API مباشر

عبر ChatGPT (الأسهل)

  • اكتب بأي لغة — عربي أو إنجليزي أو فرنسي
  • ChatGPT يُحسّن البرومبت تلقائياً
  • يمكنك المحادثة وطلب تعديلات ("اجعل الإضاءة أكثر دفئاً" أو "أزل الخلفية")
  • متاح لمشتركي ChatGPT Plus وTeam وEnterprise

عبر API (أكثر تحكماً)

  • برومبتك يذهب مباشرة لـ DALL-E 3 بدون إعادة صياغة
  • تحكم كامل في نسبة الأبعاد والجودة
  • أفضل للتطبيقات والأتمتة
  • تسعير لكل صورة: 0.04$ (1024x1024) إلى 0.12$ (1792x1024)

بنية البرومبت المثالية لـ DALL-E 3

DALL-E 3 يفهم اللغة الطبيعية أفضل من أي نموذج آخر. لا تحتاج لكلمات مفتاحية أو صياغة خاصة — اكتب كأنك تصف صورة لشخص بجانبك:

البنية المقترحة

  1. نوع الصورة: "صورة فوتوغرافية" أو "رسم توضيحي" أو "لوحة زيتية" أو "رسم بالألوان المائية"
  2. الموضوع: ما يوجد في الصورة بتفصيل
  3. البيئة: أين يحدث المشهد
  4. الإضاءة والمزاج: نوع الضوء والجو العاطفي
  5. الأسلوب الفني: مرجع فني أو تقني

مثال مُحسّن:

A watercolor illustration of a cozy bookshop on a rainy evening, warm golden light spilling from the windows onto the wet cobblestone street, a cat sitting in the window display among stacked books, autumn leaves caught in small puddles, soft impressionistic style with visible brushstrokes

  • الملصقات والإعلانات: ملصق فيلم بعنوان واضح

القيود وكيفية التعامل معها

إعادة صياغة ChatGPT

عندما يُعيد ChatGPT صياغة برومبتك، قد يُضيف تفاصيل لم تطلبها أو يُغيّر عناصر. للتحكم أكثر:

  • اطلب من ChatGPT أن يُظهر لك البرومبت قبل التوليد: "اعرض لي البرومبت الذي ستستخدمه قبل التوليد"
  • كن محدداً جداً في ما تريد وما لا تريد
  • استخدم عبارة "I NEED exactly this without any changes" عبر API

لا يوجد برومبت سلبي

DALL-E 3 لا يدعم البرومبتات السلبية مثل Stable Diffusion. بدلاً من ذلك، صِف ما تريد بوضوح. بدلاً من "بدون أشخاص"، اكتب "مشهد فارغ بدون أي شخصيات بشرية".

قيود المحتوى

DALL-E 3 له قيود صارمة على أنواع المحتوى. لا يُنشئ صوراً واقعية لأشخاص حقيقيين بالاسم، ولا يُنشئ محتوى عنيفاً أو غير ملائم.

أسئلة شائعة

ما الفرق بين DALL-E 3 عبر ChatGPT وعبر API؟

عبر ChatGPT، يُعيد ChatGPT صياغة برومبتك وتحسينه قبل إرساله لـ DALL-E 3 مما قد يُحسّن النتائج لكنه يُغيّر البرومبت الأصلي. عبر API المباشر، يمكنك إرسال البرومبت كما كتبته بالضبط بدون أي إعادة صياغة، مما يمنحك تحكماً أكبر.

هل يدعم DALL-E 3 البرومبتات بالعربية؟

نعم، عبر ChatGPT يمكنك كتابة برومبتك بالعربية بالكامل وسيترجمه ChatGPT ويُحسّنه تلقائياً قبل إرساله لـ DALL-E 3. عبر API مباشرة، الإنجليزية تعطي نتائج أفضل وأكثر دقة لأن النموذج مدرّب بشكل أساسي على بيانات إنجليزية.

كم صورة يمكن توليدها بـ DALL-E 3 يومياً؟

مشتركو ChatGPT Plus يحصلون على حد يومي يتغير حسب سياسات OpenAI وقد يختلف حسب الحمل على الخوادم. عبر API، لا يوجد حد يومي ثابت لكن هناك تسعير لكل صورة بحسب الدقة المطلوبة (من 0.04$ إلى 0.12$ للصورة).

ما أفضل طريقة لكتابة برومبت DALL-E 3 للحصول على نتائج دقيقة؟

اكتب جملاً كاملة بوصف تفصيلي كأنك تشرح الصورة لشخص. حدد بوضوح الأسلوب الفني والإضاءة والتركيب والمزاج والألوان. DALL-E 3 يفهم السياق والنية أفضل من النماذج الأخرى، لذا اللغة الطبيعية الواضحة هي الأكثر فعالية.
