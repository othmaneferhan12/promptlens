Структура промпта Midjourney

Каждый промпт Midjourney состоит из двух частей: текстовое описание и параметры. Текстовая часть описывает, что вы хотите увидеть, а параметры управляют техническими аспектами генерации.

Общая схема:

/imagine [объект], [описание сцены], [стилистические модификаторы], [освещение], [цветовая палитра] --параметры

Пять компонентов эффективного промпта

  1. Основной объект: Главный элемент изображения. Указывайте его в начале промпта, так как Midjourney придаёт наибольший вес первым словам.
  2. Описание сцены: Окружение, фон, действие. «Кафе в Париже», «космическая станция», «ночной город».
  3. Стилистические модификаторы: Медиум (фотография, масло, цифровой арт) и стиль (кинематографический, аниме, фотореализм).
  4. Освещение: Один из самых влиятельных элементов. «Golden hour», «dramatic lighting», «neon glow», «soft diffused light».
  5. Параметры: Технические настройки, управляющие генерацией.

Все параметры Midjourney v6

--ar (Aspect Ratio) — Соотношение сторон

Управляет пропорциями изображения. По умолчанию — 1:1 (квадрат).

  • --ar 16:9 — широкий формат, идеален для пейзажей и кинематографических кадров
  • --ar 9:16 — вертикальный, подходит для портретов и мобильных обоев
  • --ar 2:3 — классический портретный формат
  • --ar 3:2 — стандартный фотографический кадр

--stylize (или --s) — Стилизация

Управляет тем, насколько художественно Midjourney интерпретирует промпт. Диапазон: 0–1000, по умолчанию 100.

  • --s 0 — минимальная стилизация, максимальное следование промпту
  • --s 100 — баланс между точностью и красотой
  • --s 250–500 — высокая художественность, отлично для арта
  • --s 750–1000 — максимальная художественность, может отклоняться от промпта

--chaos (или --c) — Хаос

Управляет разнообразием результатов. Диапазон: 0–100.

  • --c 0 — все 4 варианта будут похожи друг на друга
  • --c 15–30 — умеренное разнообразие, рекомендуется для большинства случаев
  • --c 50–100 — сильное разнообразие, отлично для поиска неожиданных решений

--weird (или --w) — Странность

Добавляет необычные, сюрреалистические элементы. Диапазон: 0–3000.

  • --w 0 — стандартный результат
  • --w 250–500 — лёгкая сюрреалистичность
  • --w 1000–3000 — сильная абстракция и сюрреализм

Другие важные параметры

  • --v 6.1 — версия модели (рекомендуется v6.1 для лучшего качества)
  • --no [элемент] — негативный промпт (например, --no text, watermark)
  • --q 2 — повышенное качество (занимает больше времени)
  • --seed [число] — фиксированный сид для воспроизводимости результатов
  • --tile — создаёт бесшовный паттерн (для фонов и текстур)
  • --stop [число] — останавливает генерацию на определённом проценте (10–100)

Продвинутые техники

Мульти-промпты (Multi-prompts)

Используйте двойное двоеточие :: для разделения концепций и назначения им весов:

cyberpunk city::2 rainy night::1.5 neon signs::1 --ar 16:9 --v 6.1

Здесь «cyberpunk city» получает вес 2, «rainy night» — 1.5, а «neon signs» — 1. Это даёт тонкий контроль над композицией.

Пермутации (Permutations)

Генерируйте несколько вариантов одним командой с помощью фигурных скобок:

/imagine a {red, blue, golden} dragon in a {forest, cave, sky} --ar 16:9

Это создаст 9 разных заданий со всеми комбинациями.

Вставьте URL изображения перед текстовым промптом. Midjourney будет использовать его как стилистическую основу. Чтобы найти идеальный промпт для референсного изображения, используйте ImageToPrompt для автоматического анализа.

Частые ошибки

  1. Слишком длинные промпты. Midjourney v6 лучше работает с концентрированными описаниями. 40–80 слов — оптимально.
  2. Игнорирование --ar. Квадратный формат редко подходит. Всегда указывайте нужное соотношение сторон.
  3. Использование «не делай» вместо --no. Midjourney лучше понимает официальный параметр --no.
  4. Отсутствие освещения. Указание типа освещения — один из самых влиятельных элементов промпта.
  5. Использование промптов от Stable Diffusion. Синтаксис (термин:1.2) не работает в Midjourney. Используйте :: для весов.

Часто задаваемые вопросы

Какая версия Midjourney лучшая в 2026 году?

Midjourney v6.1 — рекомендуемая версия. Она обеспечивает лучшее качество, более точное следование промпту и улучшенную генерацию текста на изображениях. Используйте --v 6.1.

Можно ли писать промпты на русском?

Технически да, но результаты будут хуже. Midjourney обучался преимущественно на англоязычных данных. Рекомендуется писать промпты на английском. ImageToPrompt может автоматически сгенерировать англоязычный промпт из вашего изображения.

Чем отличается --stylize от --chaos?

--stylize управляет художественностью интерпретации — насколько «красиво» Midjourney пытается сделать изображение. --chaos управляет разнообразием между 4 вариантами в сетке. Они независимы и могут использоваться вместе.
