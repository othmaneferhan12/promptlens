5 Midjourney Prompt Tips That Actually Work

From --v 6.1 pros

TIP 1

Always Specify Aspect Ratio

Add --ar 16:9 for landscapes, --ar 9:16 for portraits, --ar 1:1 for square. Default is 1:1 which wastes space.

TIP 2

Use --style raw for Photos

The default Midjourney aesthetic adds painterly effects. For photorealistic output from images, add --style raw to every prompt.

TIP 3

Stack Lighting Descriptors

Terms like "golden hour", "dramatic rim lighting", "overcast diffused light" have more impact than almost any other words in Midjourney.
