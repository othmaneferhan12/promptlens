DALL-E 3

Guia Completo de Prompts DALL-E 3

Março 2026 · 10 min de leitura · DALL-E 3 · ChatGPT · OpenAI · Atualizado em março 2026

O DALL-E 3 da OpenAI é único entre os geradores de imagens de IA. Diferente do Midjourney, Stable Diffusion ou Flux, ele é profundamente integrado ao ChatGPT, o que significa que você pode descrever suas imagens em linguagem natural conversacional. Melhor ainda, o DALL-E 3 é atualmente o melhor modelo para renderizar texto legível em imagens geradas. Este guia ensina como aproveitar essas capacidades únicas.

Como Acessar o DALL-E 3

O DALL-E 3 está disponível por meio de vários canais:

  • ChatGPT Plus: 20$/mês — o meio mais simples, com interface conversacional
  • ChatGPT Enterprise / Team: Para equipes e empresas
  • API da OpenAI: Acesso programático, preço por token
  • Microsoft Copilot: Versão gratuita limitada (antigo Bing Image Creator)

O acesso via ChatGPT é recomendado para a maioria dos usuários, pois o ChatGPT atua como um assistente de prompting, reformulando automaticamente suas solicitações para otimizar os resultados.

A Abordagem de Linguagem Natural

O DALL-E 3 foi projetado para compreender linguagem natural conversacional. Você não precisa de sintaxe especial, ponderação ou parâmetros. Simplesmente escreva o que deseja ver, como se estivesse descrevendo a imagem para alguém:

A cozy reading nook by a large window on a rainy day. There's a comfortable armchair with a knitted blanket, a steaming cup of tea on a small side table, and a bookshelf filled with old leather-bound books. The light is soft and warm, filtering through rain-streaked glass. The style should be like a watercolor illustration with soft, muted colors.

Esse prompt em linguagem natural produzirá um resultado muito próximo da descrição. O DALL-E 3 compreende contexto, atmosfera e relações espaciais entre elementos melhor que qualquer outro modelo quando recebe frases completas.

Estrutura recomendada

Para resultados ideais, estruture seu prompt assim:

  1. Assunto principal: O que está no centro da imagem
  2. Contexto / cenário: Onde a cena se passa
  3. Iluminação e atmosfera: O clima desejado
  4. Estilo artístico: O visual desejado (fotografia, ilustração, pintura, etc.)
  5. Detalhes específicos: Cores, composições, elementos adicionais

Renderização de Texto nas Imagens

Esta é a força mais distinta do DALL-E 3. Diferente do Midjourney e do Stable Diffusion, que frequentemente falham ao renderizar texto legível, o DALL-E 3 lida relativamente bem com palavras curtas.

Boas práticas para texto

  • Coloque o texto entre aspas: a coffee mug with the text "Good Morning" written on it
  • Limite-se a 1-5 palavras: Além disso, os erros de renderização se tornam frequentes
  • Especifique o estilo tipográfico: in bold sans-serif letters, in elegant script font
  • Indique o posicionamento: text centered on the front of the mug

A Integração com o ChatGPT

Quando você usa o DALL-E 3 via ChatGPT, um processo de reformulação ocorre automaticamente:

  1. Você escreve sua solicitação em linguagem natural
  2. O ChatGPT interpreta sua solicitação e gera um prompt detalhado e otimizado
  3. Esse prompt reformulado é enviado ao DALL-E 3
  4. A imagem é gerada e exibida junto com o prompt utilizado

Essa reformulação é geralmente benéfica — o ChatGPT adiciona detalhes de composição, iluminação e estilo que você não especificou. No entanto, se você quiser controle total, pode pedir:

Gere uma imagem com exatamente este prompt, sem modificá-lo: [seu prompt aqui]

Dica: o modo iterativo

Uma das vantagens únicas da integração com o ChatGPT é a iteração conversacional. Após uma geração, você pode dizer:

  • "Deixe a iluminação mais quente e adicione mais névoa"
  • "Mantenha o mesmo estilo, mas substitua o gato por um cachorro"
  • "Refaça a mesma imagem, mas em formato paisagem 16:9"
  • "Gostei do estilo, mas faça mais escuro e dramático"

Política de Conteúdo

O DALL-E 3 tem a política de conteúdo mais restritiva entre os grandes geradores. Veja as limitações principais:

  • Sem personalidades reais: Impossível gerar retratos de pessoas reais nomeadas
  • Sem conteúdo violento explícito: Cenas de violência gráfica são recusadas
  • Sem conteúdo adulto: Nenhum conteúdo sexualmente explícito
  • Estilos de artistas recentes: O DALL-E 3 se recusa a copiar o estilo de artistas vivos nomeados

Para contornar a restrição sobre estilos de artistas, descreva as características visuais em vez de nomear o artista: em vez de "in the style of [artista]", descreva "with bold geometric shapes, vibrant primary colors, and thick black outlines".

Limitações e Soluções

  • Resolução limitada: O DALL-E 3 gera em 1024x1024, 1792x1024 ou 1024x1792. Use um upscaler para resolução superior.
  • Sem controle fino: Sem CFG scale, sem sampler, sem seed reprodutível. A iteração é feita de forma conversacional.
  • Reformulação automática: O ChatGPT modifica seus prompts. Peça explicitamente para não fazer isso, se necessário.
  • Consistência de personagens: Difícil manter o mesmo personagem entre gerações. Descreva os detalhes físicos com precisão.

Perguntas Frequentes

Como acessar o DALL-E 3?

O DALL-E 3 está acessível via ChatGPT Plus (20$/mês), ChatGPT Enterprise, API da OpenAI e Microsoft Copilot (versão gratuita limitada). O acesso via ChatGPT é o mais simples, pois o ChatGPT reformula automaticamente seus prompts para otimizar os resultados.

O DALL-E 3 pode gerar texto legível nas imagens?

Sim, essa é uma das forças únicas do DALL-E 3. Ele consegue renderizar texto curto (1-5 palavras) de forma bastante confiável nas imagens. Para melhores resultados, especifique o texto entre aspas no seu prompt e indique a fonte e o estilo desejados.

Por que o ChatGPT modifica meus prompts do DALL-E 3?

O ChatGPT atua como intermediário e reformula seus prompts para melhorar os resultados e respeitar as políticas de conteúdo da OpenAI. Você pode ver o prompt modificado na interface. Para controlar isso, especifique "Use exatamente este prompt, não o modifique" na sua mensagem.

O DALL-E 3 suporta prompts negativos?

Não, o DALL-E 3 não suporta prompts negativos como o Stable Diffusion. Para evitar elementos indesejados, formule seu prompt de maneira positiva ou especifique explicitamente o que você não quer no texto do prompt (ex: without any text or watermarks).
